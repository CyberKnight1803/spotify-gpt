# SpotifyGPT

Use GPT-4 to create Spotify playlists using prompts. 


## How to run?

### Environment

Create Environment:  `conda env create -f env.yml`
Activate Environment: `conda activate spotify-gpt`


### Run 
```
python main.py
```
